# TheHelloWorldProject
Hello World Programs in different languages

### Python
```
python hello.py
```

### C
```
gcc hello.c -o hello && ./hello
```
### C++
```
g++ hello.cpp && ./a.out
```

### Javascript
```
node hello.js
```

### Go
```
go run hello.py
```

###Lisp
```
clisp hello.lisp
```

###Java
```
javac hello.java
java hello
```
